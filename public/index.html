Toys Routes

#Method URL Body Description
GET /toys none Get a list of all toys.
Optional queries:
  • perPage : Select the amount of toys per page.
  • page : Select the page number you want.
    		•
    GET /toys/search?s = 'Mickey Mouse' none search for a toy by name or info
    Optional queries:
  • perPage : Select the amount of toys per page.
  • page : Select the page number you want.
    		•
    GET /toys/categories/{catname} none Get toys by category
    Optional queries:
  • perPage : Select the amount of toys per page.
  • page : Select the page number you want.
  • reverse : Select the sorting method - reverse = "yes" is ascending order; reverse = "no" is descending order.
    		•
    GET /toys/prices? min = '0' & max = '100' none Get toys by range of prices you insert
    Optional queries:
  • perPage : Select the amount of toys per page.
  • page : Select the page number you want.
  • reverse : Select the sorting method - reverse = "yes" is ascending order; reverse = "no" is descending order.
    		•
    GET /single/{id} none Get a single toy by inserting the toys id
    Optional queries:
  • perPage : Select the amount of toys per page.
  • page : Select the page number you want.
  • reverse : Select the sorting method - reverse = "yes" is ascending order; reverse = "no" is descending order.
